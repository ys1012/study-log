### 2026-03-30
- Mailサーバーの構築実習  
実習環境では暗号化なしで構築  
CUIのメールクライアントソフトはMUTT  
MUTTが暗号化されていない通信を拒否する状態  
-e "set ssl_force_tls=no; set ssl_starttls=no"　非暗号化の強制と暗号化切り替え拒否  
- 管理運用実習  
ユーザー作成でUPG自動作成  
PWDコマンドはPrint Working Directory の意味  
ユーザー削除は-rコマンド忘れずに  
サーバー証明書は~/.ssh/known_hostsに保存されている
